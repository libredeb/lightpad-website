LightPad

A lightweight, simple and powerful application launcher for Linux, compatible with both X11 and Wayland.

Download Now GitHub

Features

Lightweight

Built with Vala and GTK+ 3.0 for optimal performance and minimal resource usage.

Wayland Compatible

Works seamlessly on both X11 and Wayland display servers.

Customizable

Personalize your experience with themes and custom backgrounds.

Get Started

Install from Package

Download the pre-built package for your distribution:

Debian/Ubuntu

sudo dpkg -i lightpad_*.deb

Fedora/RHEL

sudo rpm -i lightpad_*.rpm

Build from Source

For the adventurous souls who want to build from source. The following commands are examples and may vary depending on your distribution. For detailed instructions, please refer to our official documentation:

View Official Documentation

Dependencies

sudo apt install meson valac libgtk-3-dev libgee-0.8-dev

Build Steps

meson build --prefix=/usr
ninja -C build
sudo ninja -C build install

Run It!
